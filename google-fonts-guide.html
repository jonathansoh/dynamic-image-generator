📤 Upload to cPanel

Step 1: Download a font from above (click button)
Step 2: Go to cPanel → File Manager → public_html/img/fonts/
Step 3: Upload the .ttf file
Step 4: Test Admin Panel

❓ Why Not Use Google Fonts Directly?

<link href="https://fonts.googleapis.com/css2?family=Roboto">

This works in browsers only. PHP's GD library needs actual .ttf files because it renders images on the server, not in the browser.
