🔤 Font Setup for Image Generator

Step 1: Download the font file below (Free, GPL licensed)

Open Sans Regular (Free, GPL licensed)

⬇️ Download Open Sans (GitHub) ⬇️ Download WOFF (CDNFonts)

After download: Rename to opensans.ttf and upload to fonts/ folder

Step 2: Upload to your server via cPanel
  1. Go to cPanel → File Manager
  2. Navigate to public_html/img/fonts/
  3. Click Upload
  4. Select the opensans.ttf file you downloaded
Step 3: Test the admin panel 🎨 Open Admin Panel

Alternative: Use Built-in Fonts

If you can't download fonts, the system will use built-in GD fonts (small size). To fix this:

  1. Search Google for "free arial ttf download"
  2. Download any .ttf file
  3. Rename to arial.ttf
  4. Upload to fonts/ folder

Note: The font file is needed for the font size slider to work properly. Without TTF fonts, text will be small (built-in GD fonts only).
