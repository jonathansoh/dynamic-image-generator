# Fonts Directory

This folder contains TrueType (.ttf) font files for better text rendering.

## Supported Fonts

The admin interface supports these font families:
- Arial
- Helvetica
- Georgia
- Times New Roman
- Courier New
- Verdana
- Impact

## Adding Fonts

1. Place .ttf files in this directory
2. Name them exactly as the font family (lowercase, no spaces):
   - Arial → `arial.ttf`
   - Helvetica → `helvetica.ttf`
   - Georgia → `georgia.ttf`
   - Times New Roman → `timesnewroman.ttf`
   - Courier New → `couriernew.ttf`
   - Verdana → `verdana.ttf`
   - Impact → `impact.ttf`

3. The system will automatically use these fonts if available

## Free Font Sources

- Google Fonts (convert to TTF)
- Open Font Library
- Font Squirrel

## Note

If no TrueType fonts are available, the system falls back to built-in GD fonts.
